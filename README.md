# Bond Digitally

Modern, responsive website for **Bond Digitally** — a digital systems agency that combines strategy, automation, marketing, and development.

Built with **React 19** + **Vite** and deployed on **GitHub Pages**.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📦 Build & Deploy

```bash
npm run build      # Build for production
npm run preview    # Preview production build
npm run deploy     # Deploy to GitHub Pages
```

## 🛠️ Tech Stack

- **React 19** — UI components
- **Vite 6** — Build tool
- **Vanilla CSS** — Custom design system
- **GitHub Actions** — CI/CD deployment
- **GitHub Pages** — Hosting
